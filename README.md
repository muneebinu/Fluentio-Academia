# Fluentio Academia 🎓

Fluentio Academia is a high-performance, modern web platform designed for a language learning startup. The site bridges the gap between traditional grammar rules and real-world communication confidence, specializing in English mastery and German (up to A2) instruction.

## ✨ Key Features

- **Responsive Design:** A fully fluid layout that works seamlessly across mobile, tablet, and desktop devices.
- **Interactive UI Components:** Includes a custom typewriter effect for hero sections, sticky navigation headers, and scroll-reveal animations.
- **Course Management:** Dedicated modules for Spoken English and German A2, featuring pricing comparison tables and curriculum highlights.
- **Modern Styling:** Utilizes a custom design system with CSS variables, mesh gradients, and glassmorphism elements.
- **Mobile Optimized:** Features a dedicated mobile navigation drawer with custom hamburger animations.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Modern Flexbox & Grid)
- **Interactivity:** Vanilla JavaScript (ES6+)
- **Design System:** Custom CSS Architecture with a focus on typography (Outfit & Inter fonts)
- **Integrations:** Google Forms for enrollment and contact management

## 📂 Project Structure

- `index.html`: Main landing page with hero, mission, and featured content.
- `courses.html`: Detailed breakdown of educational offerings and pricing.
- `about.html`: Brand story and core faculty profiles.
- `contact.html`: Student inquiry and social media integration.
- `style.css`: Comprehensive design system and layout logic.
- `script.js`: Core functional logic for animations and site behavior.
